# Ratified published-surface changes (check-pkg-freeze.sh). EMPTY by
# design until the first v1+ release tag exists: pre-v1.0.0 the gate is
# ADVISORY (the surface is design-fluid) and consults no allowlist.
#
# From v1.0.0 the gate enforces, and a ratified change is recorded here
# as `<baseline-sha12> <package>: <finding>` — the exact apidiff finding
# bound to the merge-base commit it was ratified against (the gate
# prints the ready-to-paste line on failure). The binding is the
# guarantee: once the ratifying PR merges, every future merge-base is a
# different commit, so an entry can never license a recurring same-text
# finding or pre-authorize a future break; superseded entries warn
# until removed. Package removals are never allowlistable
# (deprecate-then-major cycle, ADR-0069 §3).
